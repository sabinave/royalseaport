A visualization tool for overviewing the Royal Seaport Construction

In this project, results from the Royal Seaport are visualized. The user can explore the different phases of the construction but also unique buildings. The visual provides an easy and straightforward way to retrieve and discover data from several aspects. Go to Video.

How to navigate

This view shows all phases in the Royal Seaport. The yellow houses represent all the projects in this area. If one hover over one house, with the mouse, one get some information about that project. Click on the house to get even more information, shown in a picture below. On the side is a bar with different categories. All categories has their own sub-categories that the user can choose. Choosing a category will change color on the houses depending on how well that project performs in that particularly category. The hue-bar in the right corner also gives an indication on how well the projects perform. The category combined with the hue-bar is a simple tool for one wanting to easily find the house that performs the best in each category. If one hover over the houses in the hue-bar, one can see information about that project. It is also possible to click on that house to go to the more specific view with information for that project.

Phases

This page is shown when the user clicks on a phase they want to get more information about. Here, the user has clicked on Norra 1. The donut-bar shows information about the phase and how well the projects in that phase has performed.

Buildings

The view to the right is the pop-up window that shows when the user has clicked on a yellow house, representing a project. The purple bar represents the required level set from the government. It is specified on the right if it is good to perform lower or higher than that level.

Learning objectives

A lot was learnt for this project. How to gather a huge amount of information about a project and then try to understand it and visualise it in an easy way. We learned to use D3 for increased understanding and interpretation of data.
